# React Lab 4 - Blog App

## Objective

- Understand React Component Lifecycle.
- Implement `componentDidMount()`.
- Implement `componentDidCatch()`.
- Fetch data using the Fetch API.

## Lifecycle Methods

- componentDidMount()
- componentDidCatch()

https://jsonplaceholder.typicode.com/posts

## Commands

```bash
npx create-react-app blogapp
npm start
```

Successfully created a React application that fetches blog posts using the Fetch API and displays them using React lifecycle methods.
