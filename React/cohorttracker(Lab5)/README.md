# React Lab 5 - Cohort Tracker Styling

## Objective

- Learn React component styling.
- Use CSS Modules.
- Apply conditional inline styles.
- Style components using `className` and `style`.

## Technologies Used

- React
- JavaScript
- CSS Modules
- Node.js

## Files Modified

- CohortDetails.js

## Features

- Styled cohort cards using CSS Modules.
- Applied conditional heading colors:

  - Blue for **Scheduled** cohorts.
- Improved layout with bordered cards and spacing.

```bash
npm install

```

Successfully styled the Cohort Tracker application using CSS Modules and conditional inline styling in React.
